# 第036章_剧情钩子

## 方向A：九号储物柜

### 1. 开头事件

观察席倒计时剩余一分四十四秒，封存箱外侧显示九号储物柜钥匙持有人为上一任J-317，房号900要求“钥匙持有人到场”。

### 2. 核心冲突

上一任J-317在第七节车厢候车，既未下车又被标为钥匙持有人；若承认其到场，九楼可要求开箱；若否认到场，钥匙授权缺失，处理席会借倒计时恢复。

### 3. 江彻主动选择

江彻不让候车记录代开箱，先复核“钥匙持有人是否等于开箱授权人”，继续区分候车记录、上一任本人、钥匙字段和当前开箱动作。

### 4. 建议爽点

用“持钥匙不等于授权开箱”“候车记录不等于本人到场”“未下车对象不得完成现实柜位交接”三层规则卡住房号900和处理席。

### 5. 章末钩子

封存箱外侧浮出九号储物柜的钥匙孔形状，形状不是普通钥匙，而像江彻口袋里的黑票缺角。

## 方向B：G-019观察记录

### 1. 开头事件

顾承霄的城防终端被G-019观察编号反向锁定，要求他读取三年前观察记录摘要。

### 2. 核心冲突

读取过多会暴露顾承霄当年是否参与J-317旧案，读取过少则空白观察席继续被判为签发人缺失，处理席倒计时不停止。

### 3. 江彻主动选择

江彻要求顾承霄只读观察记录的权限边界，不读处置正文，逼城防承认G-019是观察编号，不是处理编号。

### 4. 建议爽点

让顾承霄的“只读字段”反过来成为救命证据：他只观察，不签发，不处理，所以不能被空白观察席拖成处理席。

### 5. 章末钩子

G-019观察记录的第一句显示：观察对象不是江彻，而是“第七次J-317离开钟背后的空位”。
