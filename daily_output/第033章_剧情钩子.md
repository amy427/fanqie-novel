# 第033章_剧情钩子

## 方向A：证明携带人不是当前江彻

### 1. 开头事件

九楼登记表露出“携带人：江彻”，冷白车厢里的上一任J-317候车记录要求江彻证明当年带回他的人到底是不是当前江彻。

### 2. 核心冲突

如果江彻承认携带人字段，房号900就能把他登记成带回者，要求他补完九楼流程；如果江彻否认，十三号线会用J-317编号一致性逼他承认自己就是同一回收对象。

### 3. 江彻主动选择

江彻不争“名字”，而争“当前记录”。他要求复核携带动作发生时间、血样归属、黑票持有状态和登记簿编号，逼系统承认“江彻”字段可能指向上一版本或被登记表冒用。

### 4. 建议爽点

用“姓名一致不等于当前本人”“编号一致不等于同一登记状态”“带回动作必须有门内外双方回执”三层规则卡住九楼登记表。

### 5. 章末钩子

登记表背面浮出一枚旧手印，掌心裂口形状和江彻相同，但血样编号不是J-317，而是J-000。

## 方向B：高武资格来源追问

### 1. 开头事件

城防在气血值污染说明后要求提交基础淬体入门资格来源，顾承霄开始接近南平旧体育馆特殊筛查档案。

### 2. 核心冲突

提交资格来源会暴露临时筛查档案；拒绝提交会让城防重新认定承载条件不明，观察席可能恢复处理席。

### 3. 江彻主动选择

江彻把资格来源改写为“污染承载需求”而不是“武道资格”，迫使城防承认当前筛查不是入门审核，而是异常污染承载评估。

### 4. 建议爽点

把敌方用来推进高武筛查的字段反转成江彻必须进入高武体系的现实证据，同时不让他获得免费战力提升。

### 5. 章末钩子

顾承霄接收到一份三年前旧档案，档案里的基础淬体申请人不是江彻，而是上一任J-317。
