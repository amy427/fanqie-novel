# 第029章_市场包装建议

## 1. 标题确认

- 草稿原题：第029章 陪同载体
- 发布确认标题：第029章 影子补票

确认理由：

1. “陪同载体”规则感强，但偏冷、偏系统术语。
2. “影子补票”更短，更有画面，能直接传达本章危险：不是人要上车，而是影子被拿去补票。
3. “别让他一个人上车”追读感最强，但更适合作为章末钩子和下一章承接，不建议提前放进本章标题。
4. “上一任J-317”信息量太大，会提前泄露章末翻牌，不适合作为本章标题。

## 2. 开头留存力

开头直接承接列车员现场复核补票单，危险目标明确，齐子衡影子先动，江彻掌心失血和旧河门牌异动同时压上来，适合保留。

## 3. 本章可卖爽点

1. 江彻把“影子愿意补票”拆成“影子没有活人资格”。
2. 江彻把“候车需持票”反打成“这不是候车，是上车”。
3. 江彻用齐子衡本人活体反应、拒绝授权和见证物在场，把补票单逼成复核问答单。
4. 章末上一任J-317在第七节车厢候车，旧案线向核心推进。

## 4. 章末追读钩子

强。

追读理由：

1. “上一任J-317”直接关联主角身份谜团。
2. “第七节车厢”与十三号线旧案形成明确下一章目标。
3. “别让他一个人上车”把危险从齐子衡影子转移到江彻与上一任江彻之间，追问感强。

## 5. 可用于简介或短视频的卖点句

1. 别人补票上车，他逼列车员先证明影子算不算人。
2. 十三号线想拿同住人的影子陪他回收，江彻直接把补票单改成审问单。
3. 影子说愿意上车，江彻问它：你有活人的疼、怕和拒绝权吗？
4. 旧车厢里坐着上一任J-317，用他的声音说：别让他一个人上车。

## 6. 读者感受判断

- 应强化：规则反制的快感、上一任J-317登场的期待、齐子衡本人和影子的身份冲突。
- 不需强化：白手套解释和术语扩展，目前信息量足够。
- 风险：如果后续第030章不立刻承接第七节车厢，章末钩子会被浪费。
