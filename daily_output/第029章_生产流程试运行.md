## 2. 主编任务确认

第029章标题：陪同载体。

开头冲突：

列车员要求齐子衡影子作为“第八次样本陪同载体”补票上车。

核心规则冲突：

1. 影子能否代表齐子衡本人。
2. 同住关系是否能转化为陪同责任。
3. 样本自动接收暂停时，陪同载体是否能先行上车。
4. 补票、登记、候车、上车四个流程是否被十三号线混用。

江彻主动选择：

江彻不直接对抗列车员，而是逼对方把“补票单”改成“复核问答单”，用活人状态、恐惧反应、拒绝授权、见证物在场来阻止影子先行上车。

爽点：

江彻把“影子愿意”拆成“乘车倾向”，再把“候车需补票”逼成“实际已上车”，抓住字段漏洞完成反制。

代价：

1. 江彻持续失血。
2. 认线压向心口。
3. 见证回执被血浸。
4. 第八次回收车厢仍未撤销陪同载体需求。

章末钩子：

问答单背面出现新乘客栏，上一任 J-317 已在第七节车厢候车，并用江彻自己的声音说“别让他一个人上车”。
