# 第021章 未完成登记的人
 柜台后的木门刚关死，九灯钟表铺外面的卷帘门就自己响了。
 不是被人拍。
 是被什么东西从另一边一下一下按住，又慢慢往里推。
 门缝下面先钻进来一滩水。
 水很黑。
 沿着地砖缝往柜台这边爬。
 江彻刚撑起半个身子，那滩水里就映出了一行门牌。
 404。
 紧接着，敲门声到了。
 “同住人。”
 “开门。”
 声音是齐子衡的。
 可尾音发黏，像有第二个人贴着他喉咙一起说话。
 唐九灯已经把唐七灯页副本攥紧，反手掀开柜台底板，拖出一只细长铁盒。
 盒子里不是刀。
 是三根磨得发亮的定位钉，一卷旧铜线，还有那只停在九点的怀表。
 “它来得比我想的快。”
 江彻看着地上那滩水，没有接话。
 他右手的小臂还在抽。
 残缺罚单压在掌心，冷得像一小块刚从冰水里捞出来的铁。
 背面的字没退。
 残缺罚单
 剩余权限：一次低阶违规判定
 使用代价：认线加深
 手机在这时亮了。
 苏见雪打来的。
 江彻按下接听。
 “说。”
 对面没有废话。
 “你那张临时筛查卡被剪掉以后，旧体育馆的体征刷新开始掉帧。”
 “封存箱刚才弹了一次。”
 “定位钉自己偏了七厘米。”
 “还有一条。”
 她顿了半秒。
 “城防的人到了外街。”
 江彻眼神沉下去。
 “顾承霄？”
 “没看见人。”
 “但待同步档案刚被人尝试拉取。”
 “查询字段是特殊筛查个体，江彻。”
 外面的敲门声又响了一下。
 比刚才更近。
 “同住人。”
 “开门。”
 这一回，卷帘门中段竟然微微鼓起来一块。
 像门外真有一张脸，正慢慢贴上来。
 唐九灯把一根定位钉直接钉进门后地砖。
 叮的一声。
 那块鼓起的铁皮立刻停住。
 “见雪。”
 江彻盯着门，声音很低。
 “齐子衡现在是什么状态。”
 键盘声从电话那头传来。
 苏见雪显然已经把旧体育馆那台筛查终端又强行拉起来。
 几秒后，她开口。
 “两组记录在重叠。”
 “一组是404住户。”
 “一组是待补登。”
 “还没写死。”
 江彻听懂了。
 齐子衡还没完成登记。
 所以唐七灯页副本背面的条件，不是诱饵。
 是真的。
 地下钟室下层入口，要的就是这种人。
 外面的齐子衡像也听见了这边的安静。
 门外传来一阵发闷的笑。
 不是他平时那种轻佻。
 更像有人借着他的嘴，学会了怎么笑。
 “江彻。”
 “他们要把我写完。”
 “你再不开门，我就真成404了。”
 唐九灯低声骂了一句，手上动作却没停。
 他把旧铜线一圈圈绕上门框，又把停在九点的怀表压在柜台边缘，最后把那截烧焦红绳系到木门把手上。
 “我能给你做一扇错门。”
 “三十息。”
 “再久，这只怀表就保不住了。”
 江彻看了他一眼。
 “代价呢。”
 唐九灯头也没抬。
 “我师兄留下的最后一个定门物，废掉。”
 “还有这铺子，今晚之后多半藏不住了。”
 这不是客气话。
 旧河公寓已经开了。
 再把地下钟室下层入口从九灯钟表铺里掀出来，这里等于被彻底写进账里。
 可唐九灯还是做了。
 江彻收回目光。
 卷帘门外，水已经漫过第一根定位钉。
 门缝里慢慢塞进来一张纸。
 白纸黑字。
 住户催返通知
 目标：江彻
 关联住户：齐子衡
 当前状态：夜间催返中
 未返后果：同住关系强制履行
 纸边还挂着一截湿指甲。
 像是谁刚从门板另一边硬抠进来的。
 江彻蹲下，看了一眼纸的背面。
 背面有鞋印。
 很浅。
 但鞋尖已经越过门槛半寸。
 他又抬头看卷帘门中段。
 那块鼓起的铁皮下面，正慢慢探进来一小截灰色袖口。
 外面的东西，不只是敲门。
 它已经把“住户委员会只在门外说话，不会进入房间”这条踩了。
 江彻掌心那张残缺罚单，忽然更冷了一层。
 像在催他。
 唐九灯显然也看见了那截袖口。
 “能判？”
 江彻没马上答。
 他先对电话那头说：
 “见雪，把齐子衡那两组状态念给我。”
 苏见雪直接开口。
 “住户融合中。”
 “待补登。”
 “未完成登记。”
 外面的齐子衡像被什么东西掐了一下，声音忽然变了形。
 “别念。”
 “别让它听见。”
 江彻抬手，把手机开了免提，扔到柜台上。
 然后他站起来，朝卷帘门走了一步。
 “齐子衡。”
 门外安静了两秒。
 “你还怕死吗。”
 那边很久没有回。
 再开口时，声音终于像齐子衡自己了。
 “怕。”
 “我怕得要命。”
 “江彻，它们要把我写成门里的人。”
 “我不想只剩一张房号。”
 这句够了。
 他还有人味。
 也说明他不是单纯来骗门的壳子。
 江彻看着卷帘门下那张催返通知，忽然开口。
 “唐九灯。”
 “给我开半扇。”
 唐九灯手一顿。
 “你确定？”
 “确定。”
 “它进来怎么办。”
 江彻把残缺罚单翻到正面。
 “那就让它自己咬自己。”
 唐九灯没再问。
 他一把拽紧铜线，另一只手狠狠按住停在九点的怀表。
 怀表表盖弹开。
 里面那根细针竟反着走了半圈。
 柜台后的木门立刻发出一声轻响。
 门缝底部往外渗出一道很薄的黑线。
 像有人在地上画了一扇门。
 错门成了。
 与此同时，卷帘门被唐九灯拉起一道只能容一人弯腰挤过的缝。
 缝一开，外面的走廊就不见了。
 门外站着的不是齐子衡一个。
 他半边身子还像人，脸色白得发灰，左眼里倒映着404门牌。
 另一边肩膀后，却站着三个穿深灰制服的女人。
 嘴被细线缝着。
 每人手里都拿着一本查房册。
 她们没有完全站在门外。
 最前面那个，左脚已经踩进了门槛里。
 江彻没有给她缩回去的机会。
 残缺罚单直接抬起。
 他声音不高。
 却字字咬死。
 “旧河公寓住户须知。”
 “第四条。”
 “住户委员会只在门外说话，不会进入房间。”
 “你越线了。”
 那女人的缝线嘴猛地绷紧。
 她像是想退。
 可已经来不及。
 江彻刚才看的不是影子。
 不是气息。
 是规则落点。
 纸背鞋印，门槛湿痕，探进来的袖口，踩进来的左脚。
 全是违规成立的证据。
 他五指一合，把残缺罚单按向那只越过门槛的脚。
 罚单表面立刻浮字。
 违规对象：住户委员会
 违规事项：入户查房
 判定方式：成立
 执行权限：残缺生效
 下一秒，江彻右臂整条认线一起抽紧。
 疼得他指尖都发木。
 黑线从小臂又往上窜了一截。
 像有人把一根烧红的针顺着血管捅进去。
 可门外那三个女人同时僵住了。
 她们手里的查房册哗啦翻页。
 走廊两侧本来紧闭的房门，竟在这时一扇接一扇弹开。
 一张张长在门板上的脸同时睁眼。
 “请出示查房权限。”
 “请出示查房权限。”
 “请出示查房权限。”
 声音一层压一层。
 住户委员会第一次被查房。
 这就是残缺罚单还能做的事。
 不是杀伤。
 是让规则承认它自己犯了低阶错误，然后把错误反咬回执行体身上。
 最前面那个女人的查房册自己翻到末页。
 纸页上浮出一行黑字。
 本次查房失败
 原因：委员会越线入户
 处罚：暂时退离当前门
 她整个人被门外那股潮湿的风狠狠往后一拽。
 细线缝着的嘴一下裂开半寸。
 后面两个也跟着往后滑。
 齐子衡就夹在她们中间，被一起带歪。
 江彻等的就是这一瞬。
 他一步欺上去，左手直接扣住齐子衡的衣领，把人硬拽进门缝。
 齐子衡胸口冰得吓人。
 衣服里面却鼓着一块硬纸似的东西。
 江彻一把按上去。
 隔着布料，他摸到了一枚凸起的湿字。
 未登。
 果然没写完。
 齐子衡刚被拽进来半步，外面那三个女人就同时尖声开口。
 “住户不得长期独居。”
 “同住人不得带离住户。”
 “未完成返房，不得转入其他登记点。”
 唐九灯已经把铜线绕紧到极限。
 柜台边缘那只怀表咔的一声，裂了。
 表盘上的九点彻底碎成两半。
 “江彻。”
 “三十息没了。”
 “你要进就现在进。”
 江彻没有立刻动。
 他先把齐子衡按到地上。
 “看着我。”
 齐子衡眼里的404影子还在晃。
 可他还是本能地盯住了江彻。
 “你想不想活。”
 “想。”
 “那就别装住户。”
 “你现在不是404。”
 “你是未完成登记的人。”

 像有什么东西在和这句话抢他的嘴。

 “我是住户。”
 “住户要返房。”
 江彻抬手，一巴掌直接扇在他脸上。
 很重。

 他刚才的行为，本身就在证明他还没被写死成404。

 最后一个“层”字还在发黑，像随时会写完，也随时会断。

 “我不下去。”
 “下面有人一直敲我名字。”
 “他们说写完我，我就不会怕了。”

 “怕就对了。”
 “怕，说明你还没写完。”
 外面的住户委员会已经重新稳住。
 走廊深处，又有新的敲门声传来。
 这一回，不止一个方向。

 像整层楼都知道这里藏了一个未完成登记的人。

 “顾承霄到了。”
 “他看见特殊筛查档案了。”
 “还拿走了一份待同步记录。”
 “上面有九灯钟表铺附近的定位偏差。”

 旧体育馆线顶不住了。
 城防很快就会顺着这条偏差摸过来。
 唐九灯显然也听见了。

 “今晚真热闹。”

 地下钟室下层。

 残缺罚单已经彻底化软，像一张泡坏的旧纸，再没有第二次机会。
