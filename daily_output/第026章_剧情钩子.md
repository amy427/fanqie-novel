# 第026章 剧情钩子

## 方向A：继续当前主线

### 1. 开头事件

白手套从404门内伸出，拿着旧河公寓见证回执副页，要求江彻补齐签收。副页和江彻手中的正页互相吸引，签收栏开始自动拼合。

### 2. 核心冲突

江彻必须阻止回执拼合后直接把自己写成签收人，同时还要防止齐子衡影子趁白手套接管流程时完成返房。白手套不受住户委员会规则完全限制，普通旧河公寓漏洞不能直接压住它。

### 3. 江彻主动选择

江彻主动把问题从“谁签收”转成“谁有资格接管签收”，用顾承霄旧案观察牌、苏见雪现实端记录、唐七灯拒签事实，逼白手套说明它当年接管的是旧河公寓流程，还是十三号线回收流程。

### 4. 建议爽点

江彻让白手套在两本账之间自证权限：如果它承认旧河公寓权限，就要接受见证人拒签未闭合；如果它承认十三号线权限，就不能直接收旧河公寓回执。

### 5. 章末钩子

白手套短暂卡住后，回执副页背面浮出另一行三年前被删掉的签收人姓名，不是顾承霄，也不是韩魁，而是苏见雪。

## 方向B：插入另一条正在压近的既有线索

### 1. 开头事件

旧体育馆现实端警报被顾承霄强行降级，表面是封锁，实际是他亲手隔离待同步档案，阻止更高层自动接管。

### 2. 核心冲突

苏见雪必须判断顾承霄是在保护证据、封存证据，还是准备把江彻和旧体育馆记录一起处理掉。她不能离开旧体育馆，否则江彻的现实端核验锚点会断。

### 3. 江彻主动选择

江彻让苏见雪不要争夺终端控制权，而是读取三年前污染样本与当前体征刷新之间的同源反应，让顾承霄无法直接把当前江彻判成异常替身。

### 4. 建议爽点

现实端和副本端双线互咬：江彻在下层卡白手套签收，苏见雪在旧体育馆卡顾承霄封档，逼官方记录承认“活体反应存在”。

### 5. 章末钩子

旧体育馆封存箱自己弹开一条缝，里面三年前韩魁污染样本旁边，多出一张新标签：J-317，第八次回收前样本。
