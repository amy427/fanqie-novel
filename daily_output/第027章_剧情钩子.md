# 第027章 剧情钩子

## 方向A：继续当前主线

### 1. 开头事件

旧体育馆封存箱被白手套隔空敲响，苏见雪面前弹出“封存见证人确认留置”提示，江彻在下层听见同样的敲击声从手机里传来。

### 2. 核心冲突

苏见雪如果确认留置，见证回执正页会被白手套合法扣押；如果拒绝确认，封存箱里的三年前样本可能被判为拒签异常，反向污染旧体育馆现实端核验锚点。

### 3. 江彻主动选择

江彻不让苏见雪直接争夺终端，而是让她读取封存箱内样本与韩魁协办链异常之间的同源反应，逼白手套说明“封存见证人”是见证样本，还是见证签收。

### 4. 建议爽点

江彻和苏见雪双线反制：江彻在下层卡住证据留置点，苏见雪在现实端卡住封存见证字段，逼白手套承认它不能让见证样本替代见证人签收。

### 5. 章末钩子

封存箱里三年前韩魁污染样本旁边，多出一张新标签：J-317，第八次回收前样本。标签日期不是三年前，而是今晚。

## 方向B：插入另一条正在压近的既有线索

### 1. 开头事件

顾承霄强行降级旧体育馆警报，表面像是在封锁苏见雪，实际切断上层自动接管，让封存箱暂时留在人工观察模式。

### 2. 核心冲突

苏见雪必须判断顾承霄是在保护证据、留江彻当样本，还是准备把她和旧体育馆一并处理。顾承霄不救人，但他也不想让白手套拿走完整证据链。

### 3. 江彻主动选择

江彻让顾承霄不要再回答白手套相关问题，只让他读系统原始字段，避免被下层算成代签确认，同时把顾承霄继续钉在现场核验链里。

### 4. 建议爽点

顾承霄被迫做一次“不救人但也不让流程闭合”的选择，江彻借这个选择把官方监察从压迫源变成临时证据锚点。

### 5. 章末钩子

顾承霄在终端里看到一条被三年前删除的备注恢复：若封存见证人拒绝留置确认，则转入下一任J-317样本接收。
