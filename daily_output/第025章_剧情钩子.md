# 第025章 剧情钩子

## 方向A：继续当前主线

### 1. 开头事件

齐子衡影子里的404门牌完全抬头，影子不再贴地，而是像一扇薄门一样立在江彻和齐子衡身后，开口要求同住人回应。

### 2. 核心冲突

江彻必须在三件事之间取舍：带走旧河公寓见证回执、保住齐子衡未完成登记状态、阻止影子404把自己拖进门内。任何一项处理慢了，回执都会触发签收，或者齐子衡影子会完成住户化。

### 3. 江彻主动选择

江彻主动把七灯铜扣压在影子门牌上，尝试用唐七灯拒签见证人的旧规则，证明影子404不是齐子衡本人，不能替齐子衡完成同住人确认。

### 4. 建议爽点

江彻借“见证人拒签”反制影子404，让影子第一次被迫回答自己是否具备本人确认资格，从而暴露404补登流程并不承认影子是完整住户。

### 5. 章末钩子

影子404被短暂卡住后，门缝里露出另一只手。那只手戴着白手套，拿着三年前见证回执的另一半。

## 方向B：插入另一条正在压近的既有线索

### 1. 开头事件

旧体育馆现实端警报升级，苏见雪发现顾承霄已拿到待同步档案，并要求城防封锁旧体育馆器材通道和九灯钟表铺附近偏差点。

### 2. 核心冲突

现实端封锁会切断江彻和下层之间的临时缓冲。一旦旧体育馆隔离观察模式停止，旧河公寓就能重新确认江彻不在临时筛查归属中，直接把他补回404同住人链条。

### 3. 江彻主动选择

江彻远程让苏见雪不要和顾承霄硬碰，而是把旧体育馆封存箱里的体征刷新记录念进通话，让现实端记录和地下见证回执形成二次冲突。

### 4. 建议爽点

江彻把现实端官方档案和副本内见证回执互相咬死，逼顾承霄不能直接销毁证据，也不能继续装作只是观察者。

### 5. 章末钩子

顾承霄在旧体育馆终端上看到三年前被删除的另一行字：见证人拒签后，白手套接管签收。
