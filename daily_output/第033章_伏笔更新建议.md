# 第033章_伏笔更新建议

1. “携带人：江彻”不再直接等同当前江彻，已拆成历史携带字段、当前本人补认、J-317回收编号和旧河登记回执。
2. 上一任J-317的旧手印来源被限定为回收凭证转录，不是完整本人手印。
3. 三年前基础淬体入门资格申请人显示为上一任J-317，状态为已接收未入档。
4. J-000首次作为血样编号、黑票母本转录和担保字段出现。

3. 三年前上一任J-317为何申请基础淬体入门资格，申请为何已接收未入档。
4. “携带人已到场”指向谁，是否说明当前江彻只是被房号900等来的对象。

2. 不解释黑票母本全机制。

4. 不让上一任J-317脱离候车记录状态。
