1. 第八次回收流程代表什么：从样本倒计时推进到自动接收暂停并转入现场执行端复核，建议回收时间：短期，1到5章内继续推进。
2. 齐子衡影子404线：影子从代返房、代授权升级为可能被十三号线复核为陪同载体，建议回收时间：短期，1到5章内回收阶段结果。
3. 顾承霄是否想让J-317活：顾承霄继续只读字段、不直接救人，但主动维持人工观察，建议回收时间：中期，6到20章内继续观察。
4. 唐七灯剩余认知消耗：本章未调用唐七灯发声，而是使用铜扣见证物属性，建议回收时间：短期，1到5章内标记其暂时未被进一步消耗。
5. 白手套流程：白手套在授权链被卡后把流程交给列车员现场复核，建议回收时间：短期，1到5章内继续推进。

## 3. 本章已回收伏笔

1. 第八次样本倒计时能否直接阻止：已阶段性回收，结论为不能取消，但可通过未授权采集、字段空白、容器不规范和城防追责字段暂停自动接收。
2. 残缺罚单碎毁后的限制：已阶段性回收，江彻不能再执行低阶违规判定，只能依靠证据链和规则自相矛盾拖住流程。

## 4. 本章新增角色状态变化

1. 江彻：样本自动接收暂停，但仍被采集流程锁定，失血和心口认线压力继续存在。
2. 苏见雪：将空管标记为未授权采集并转入污染物隔离，继续维持旧体育馆人工观察锚点。
3. 顾承霄：因降级自动处置触发城防内部追责，仍选择只读字段协助卡住流程。
4. 齐子衡：本人未授权影子代签，但影子被列车员点名为复核对象。
5. 白手套：自动接收被暂停，退到列车员现场复核之后。

## 5. 本章新增规则或限制

1. 未授权采集标记不能直接覆盖白手套流程，但能让样本来源带争议。
2. 旧体育馆人工观察模式下，新增样本不能自动归档或自动移交。
3. 关联住户或同住关系不能无限代签，尤其在住户本人未完成登记且影子不具备完整自主认名时。
4. 无刻度、无批号、无封签号的空管不能被规范样本库直接接收，只能被争议或污染物隔离处理。
5. 自动接收暂停后，十三号线现场执行端可转入人工复核。
