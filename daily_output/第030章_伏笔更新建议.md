# 第030章_伏笔更新建议

## 1. 本章新增伏笔

1. 观察席十二分钟后失效。
   - 建议回收时间：短期，1到2章内。
2. 唐七灯成为第八次样本陪同席候选人。
   - 建议回收时间：短期，1到3章内。
3. 上一任J-317候车记录是否在吞噬被记录的人。
   - 建议回收时间：中期，6到20章内。
4. 第七节车厢观察席与处理席的转换条件。
   - 建议回收时间：短期，1到5章内。

## 2. 本章推进的旧伏笔

1. 上一任J-317：从“第七节车厢已候车”推进为“候车记录具备声音和诱导能力，但未必是完整本人”。
2. 顾承霄观察线：确认他的观察记录可能被旧案兼容流程转成处理席。
3. 第八次回收陪同载体：确认陪同席空缺会造成观察席转处理席的风险。
4. 唐七灯地下钟室线：唐七灯被第八次陪同席点名，地下钟室下层有敲门响应。

## 3. 本章已回收伏笔

1. “别让他一个人上车”的第一层含义已回收：若陪同席空缺，观察席会自动转为处理席，顾承霄记录会被推成清除流程。
2. 第029章观察席票面钩子已回收：顾承霄观察席实际隐藏处理席。

## 4. 本章新增角色状态变化

1. 江彻：认线进一步刺向心口，靠现场记录暂存观察席争取十二分钟。
2. 顾承霄：被十三号线旧案兼容流程试图夹成处理席，暂时脱身但风险升级。
3. 齐子衡：继续作为本人在场证明，握住七灯铜扣，影子仍被卡在待复核状态。
4. 唐七灯：从地下钟室下层状态被推到第八次陪同席候选人。
5. 上一任J-317：候车记录身份加深，真实主体仍不明。

## 5. 本章新增规则或限制

1. 观察席若空缺，可能被处理席接管。
2. 观察席需要可记录主体，不一定必须是官方观察员本人。
3. 现场记录可以暂存观察字段，但无清除权限。
4. 现场记录观察席有效时间只有十二分钟。
5. 陪同席空缺会继续逼迫流程寻找候选人。
