# 第032章_剧情钩子

## 方向A：追入房号900

### 1. 开头事件

红绳缠住江彻掌心伤口，房号900湿纸贴在他手上，地下钟室下层门打开一条缝。毛笔女人说旧河公寓没有九楼，九楼住着没被带回来的人。

### 2. 核心冲突

江彻若进入门缝，可能被房号900登记为“带回者”；若不进入，观察席倒计时耗尽后顾承霄记录会重新转向处理席，唐七灯陪同席候选也会再次被确认。

### 3. 江彻主动选择

江彻先不跨门槛，而是要求复核“未开门”和“带回”的先后关系，逼房号900承认未开门状态下不能登记带回结果，再让齐子衡以未完成登记人身份继续敲门，问出九楼到底缺谁。

### 4. 建议爽点

用“未开门不能完成带回”“响应不等于入住房间”“九楼不是正式楼层，不能套用普通住户登记”三层字段，卡住房号900直接吞人的流程。

### 5. 章末钩子

门缝里递出第一张九楼登记表，上面已有一行旧名字：上一任J-317，状态为已带回未下车。

## 方向B：气血值提交危机

### 1. 开头事件

城防终端要求顾承霄在观察席失效前提交江彻当前气血值，南平旧体育馆特殊筛查档案开始被同步调用。

### 2. 核心冲突

提交0.74会暴露江彻现实端筛查档案；不提交则处理条件不明，观察席可能恢复处理席。与此同时，房号900红绳持续牵引江彻血样。

### 3. 江彻主动选择

江彻要求苏见雪只提交“气血值无法稳定读取，原因：下层牵引污染”，把气血值从能力证明改成污染证据，迫使城防暂缓使用该数值做处置依据。

### 4. 建议爽点

把敌方要求的气血值反过来变成“样本被污染，不能作为处理依据”的证据，同时自然推进高武线和认线承载压力。

### 5. 章末钩子

城防终端接受污染说明后，弹出替代检测项：请提交J-317基础淬体入门资格来源。
