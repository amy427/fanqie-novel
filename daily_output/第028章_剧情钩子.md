# 第028章_剧情钩子

## 方向A：继续当前主线

### 1. 开头事件

十三号线进站广播从404门缝深处响起，旧体育馆封存箱内的第八次回收前样本空管开始倒计时，江彻掌心的伤口被迫继续滴血。

### 2. 核心冲突

如果江彻继续流血，样本会补齐，十三号线可借样本重启第八次回收；如果强行止血，旧河公寓见证回执正页可能失去现实端活体冲突，重新被白手套副页拉走。

### 3. 江彻主动选择

江彻不直接堵伤口，而是要求苏见雪把空管标记为“未授权采集”，同时让顾承霄读取城防系统的样本采集授权链，逼白手套说明第八次样本采集属于旧河登记、十三号线回收，还是静默处置科旧案补采。

### 4. 建议爽点

江彻利用“采集对象未同意、采集端不在现场、采集授权链缺失”三项，逼第八次样本倒计时暂停，但不能彻底取消，只能改为人工复核。

### 5. 章末钩子

样本采集暂停后，十三号线没有离开，而是在404门缝深处打开车门。列车员递出一张补票单，目标不是江彻，而是齐子衡的影子。

## 方向B：插入另一条正在压近的既有线索

### 1. 开头事件

顾承霄降级旧体育馆警报后，城防系统内部出现追责提示，要求他说明为何将J-317相关样本从自动处置改为人工观察。

### 2. 核心冲突

顾承霄如果解释，会被旧河下层算成继续代签；如果不解释，城防系统会自动恢复上层接管，苏见雪和封存箱会被一起锁死。

### 3. 江彻主动选择

江彻让顾承霄只读“追责字段”和“授权链缺失字段”，不读处置意见，把城防追责也变成白手套无法闭合的第三方证据锚点。

### 4. 建议爽点

顾承霄被迫把官方系统的追责压力反向压到白手套身上，证明白手套第八次样本接收没有完整授权链。

### 5. 章末钩子

城防终端恢复一条三年前删除的处置意见：若第八次样本采集失败，则执行“现场回收”，执行端显示为列车员。
