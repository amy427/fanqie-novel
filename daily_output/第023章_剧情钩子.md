# 第023章 剧情钩子

## 方向A：继续当前主线

### 1. 开头事件

江彻刚看到自己名字重新出现在“未完成登记名单”里，柜中更深处就开始补写状态；齐子衡胸口残缺的404再次发热，下层执行体要趁名单未合页前把两人一起归档。

### 2. 核心冲突

江彻必须在“先保住自己不被重新写入”与“继续利用齐子衡挖出唐七灯所在格位”之间二选一，任何迟疑都可能让名单完成补写。

### 3. 江彻主动选择

江彻主动舍弃一部分临时身份痕迹，优先锁住齐子衡“姓名未尽”的状态，再反向利用下层名单找唐七灯的真实位置。

### 4. 建议爽点

江彻抓住“代签人不得代答”的限制，让下层执行体自己说漏嘴，从它们的补写话术里反推出唐七灯所在柜位。

### 5. 章末钩子

唐七灯的柜位被找到，但柜门里面挂着的不是人，而是一张写着顾承霄旧编号的湿纸牌。

## 方向B：插入正在压近的既有线索

### 1. 开头事件

顾承霄在门外没有立刻强攻，而是继续通过旧体育馆终端调三年前 J-317 的删改记录；苏见雪则发现有人在反查她当年参与过的旧案封存表。

### 2. 核心冲突

如果顾承霄掌握的旧案记录与下层名单能互相印证，江彻就不再只是被旧河公寓追缴，而是会被官方和旧案观察流程同时锁定。

### 3. 江彻主动选择

江彻主动把名单里“顾承霄”的一行念出来，逼顾承霄当场表态自己究竟是观察者、代签人，还是曾经的未完成登记对象。

### 4. 建议爽点

江彻把顾承霄也拉进规则问答，利用名单和旧体育馆终端的双重记录，让这个一直站在门外的人第一次必须按规则回答。

### 5. 章末钩子

顾承霄报出一个只有三年前那次注销现场参与者才会知道的名字，而那个名字本该随着第七次 J-317 一起烂在钟背后。
