# 第024章 伏笔更新建议

## 1. 本章新增伏笔

1. 齐子衡影子里的404门牌：建议短期回收，1到5章内处理影子是否能代替齐子衡履行住户义务。
2. 七灯铜扣：建议短期回收，1到5章内说明是否可稳定唐七灯残存认知或作为下一次牵引物。
3. 旧河公寓见证回执能否带出下层：建议短期回收，1到5章内处理回执离开下层的签收代价。
4. 顾承霄承认“未闭合项”后的现实后果：建议中期回收，6到20章内影响城防系统和顾承霄对江彻的态度。

## 2. 本章推进的旧伏笔

1. 三年前江彻是否真正回来过：唐七灯残存确认当年见证对象具备活人特征，建议短期继续推进。
2. 静默处置科替身处理流程：见证回执显示唐七灯拒签后仍被替身处理，建议中期回收具体下令者。
3. 顾承霄到底知道多少：旧案观察牌显示他是代签人，建议中期回收他三年前的实际权限和动机。
4. 唐七灯为何被困下层：本章明确他是拒签见证人，建议短期稳定其残存认知。
5. 齐子衡被拖入404后会变成什么：房号开始转入影子，建议短期处理影子入门危机。

## 3. 本章已回收伏笔

1. 唐七灯并非普通受害者编号，而是三年前见证人。
2. 空钟柜中存放的关键不是唐七灯本人，而是见证回执和三年前拒签证据。
3. 顾承霄旧案观察牌的用途得到阶段性解释：它是代签人凭证，可被下层规则核验。

## 4. 本章新增角色状态变化

1. 江彻：认线越过上臂，钟背钥匙继续损坏，持有旧河公寓见证回执和七灯铜扣。
2. 齐子衡：身体房号暂未写死，但404最后一笔转入影子，影子疑似成为可独立进门的住户化对象。
3. 顾承霄：被规则逼入问答，承认旧案存在未闭合项，不能再完全站在门外观察。
4. 唐七灯：残存认知进一步消耗，但完成关键见证确认。

## 5. 本章新增规则或限制

1. 旧案观察牌不可被随意转用于非原对象补登，必须经持牌人确认。
2. 未完成登记者可作为下层通行牵引，但这会让其自身继续承担被写死风险。
3. 见证回执离开下层会触发签收栏补写，可能把取证者补入未完成登记名单。
4. 见证人不得入柜，说明空钟柜保存的是见证结果和证据，不是见证人本人。
