# Rules State

Current version date: 2026-05-06

## Confirmed Active Rules

1. 十三号线补票流程不能绕过复核对象的乘客资格。
2. 同住关系可以被十三号线尝试转换为陪同责任，但责任不等于载体。
3. 齐子衡影子没有独立活体反应、血样、疼痛反馈、自主身份签名。
4. 影子不能在本人明确拒绝授权时直接代签补票。
5. 样本自动接收暂停时，陪同载体不能被无说明地先行上车。
6. 补票单可以被现场证据链逼成复核问答单。

## Active Unknowns

1. 第八次回收车厢为什么需要陪同载体。
2. 上一任J-317是否具备独立乘客资格。
3. 第七节车厢是否属于正式乘车区、候车区，还是问询区。
4. 列车员在现场复核中为什么可限制白手套。
5. 顾承霄追责链的上级监察端是否与三年前旧案同源。
