# 《高武：违规者才配活着》长篇控制设定表

作者笔名：饮血独行

当前定位：都市男频，高武，规则怪谈，副本悬疑，升级流，阶层逆袭

核心目标：番茄小说长篇连载，前20章强留存，200章以上可扩展，具备短剧和漫剧改编感。

当前主角：江彻

当前创作原则：不靠空泛情绪词，不靠套路震惊，不靠机械爽点重复。正文要靠规则、空间、动作、判断、代价和具体后果推进。

---

# 0. 当前写作硬规则

## 0.1 格式规则

1. 人物对白、广播、对讲机、诱导声、模仿声，必须加双引号。
2. 非对白叙述不加双引号。
3. 不使用破折号。
4. 规则、站名、编号、档案、名单，单独列出。
5. 正文段落保持短，一段尽量只完成一个动作、判断或信息点。
6. 不使用明显模板化句式。
7. 不写成说明文，不写成大纲扩写腔。

## 0.2 语言规则

少写：

空气仿佛凝固。
命运的齿轮开始转动。
所有人都震惊了。
一股前所未有的力量涌上心头。
他眼神冰冷。
他深吸一口气。

多写：

具体物件。
具体声音。
具体灯光。
具体空间变化。
具体规则后果。
具体身体反应。
具体行动选择。

## 0.3 单章规则

每章约4000字。
每章至少包含以下四项中的三项：

1. 新信息
2. 新冲突
3. 新收益或新损失
4. 新悬念

每章必须有具体章末钩子。
每3章至少一次明显反转。
每5章至少一个可被读者复述的高能点。
每10章要有阶段性揭示或阶段性高潮。

# 1. 世界观设定表

## 1.1 基础世界观

灵潮之后，城市每晚都有概率出现异常规则区域。

异常区域通常以现代都市空间为外壳，例如地铁、旧公寓、医院、学校、商场、电梯、仓库、码头、地下通道。

普通人看到的是事故、设备异常、封控、失踪。

真正懂规则的人知道，这些地方是静默事件。

## 1.2 静默事件

静默事件不是单纯自然灾害。

它包含：

1. 空间异常
2. 规则污染
3. 伪规则诱导
4. 人员筛选
5. 回收机制
6. 遗留物产生
7. 规单流通

静默事件会被上层组织利用。底层人经常被送进去试错，用命踩出真规则和假规则。

## 1.3 规则结构

每个异常区域通常存在三层规则。

第一层：表层规则

写在墙上、规单上、广播里、通知里，普通人能看见。
其中部分是真的，部分是假的。

第二层：第二列字

只有江彻通过黑票能看见部分内容。
第二列字不是完整外挂，而是规则背后更接近真实的一层提示。
每次看见第二列字，都有代价。

第三层：真正源规则

目前未完全揭示。
可能与十三号线、旧河公寓、裁钟、白手套、静默处置科有关。

## 1.4 城市阶层

下城区：

底层工人、临时工、黑市、小帮派、规则贩子、试错者。大量普通人被拿来踩规则。

中城区：

城防系统、静默处置科、家族外线、规单交易中介、武道训练资源入口。

上层区域：

尚未正式展开。推测掌握完整规则档案、静默事件源头、异常资源分配权。

## 1.5 当前已出现组织

### 裁钟

目前定位：神秘组织或上层外线系统之一。

已知信息：

1. 收人、验人、分类。
2. 与邵文海有关。
3. 与试错费、旧河公寓规单有关。
4. 白手套可能在其流程上，或与其合作。
5. 韩魁只是这条线上的低层协办者。

后续功能：

代表上层利用异常区筛选和收割底层的力量。

### 白手套

目前定位：流程执行者或某一类回收系统代号。

已知信息：

1. 十三号线车厢内出现过白手套检票者。
2. 韩魁提到现实中的白手套作为上级联系人。
3. 三年前J-317旧案音频中，可能有白手套或同层人物下达替身处理命令。
4. 白手套不一定是单个人，也可能是职位、代号、异常执行体。

后续功能：

连接现实处置系统和异常回收系统。

### 静默处置科

目前定位：官方系统中的异常事件处置部门。

已知信息：

目前定位：城市官方武装与监察系统。

# 2. 主角能力表

## 2.1 主角基本信息

姓名：江彻

身份：下城区七号线夜班临时巡检，现已被官方活人库注销。

当前状态：活人名单无此人，编号J-317，已被二次标记或多次回收流程追踪。

核心人格：冷静、敏锐、会判断、记仇、求生欲强，行动先于情绪。

不写成：热血嘴炮型、圣母型、频繁自我感动型、靠运气乱莽型。

## 2.2 当前核心能力

### 黑票感应

来源：十三号线事件中获得的黑色车票。

表现：

1. 靠近异常规则时发热。
2. 遇到关键规则、遗留物、异常空间时出现感应。
3. 能引出部分第二列字。
4. 能让江彻发现隐藏规则线索。

### 第二列字

表现：

1. 在表层规则右侧或背后浮出更真实的一层内容。
2. 有时是补充，有时是反转，有时是警告。
3. 不是完整答案，必须靠江彻判断。

用途：

1. 分辨真规则和伪规则。
2. 找到生路。
3. 找到隐藏入口。
4. 发现敌人设下的规单坑。
5. 建立江彻与普通试错者的差距。

限制：

1. 每次看见第二列字，黑票污染都会加深。
2. 会形成认线。
3. 会让副本、镜面、玻璃、屏幕更容易找到江彻。
4. 不能频繁无成本使用。
5. 不能变成万能解题器。

## 2.3 污染与代价

当前污染名称：认线。

表现：

1. 手背虎口附近出现细黑线。
2. 黑线向手腕延伸。
3. 头痛、太阳穴跳痛。
4. 情绪发空。
5. 镜面中的自己可能先动。
6. 玻璃中可能映出十三号线。
7. 电脑屏幕可能出现雪花窥视。
8. 副本可能逐渐主动找上他。

苏见雪诊断：

认线不是普通伤口污染，而是顺着认知走。
每次看见不该看见的规则，就会被更深地记录。

压线剂：

作用：暂时压制认线。
代价：二十四小时内再强行看第二列字，反冲会更重。
限制：只能压制，不能根除。

## 2.4 当前能力边界

江彻目前不是强战力主角。

当前优势：

1. 判断力强。
2. 观察细节准。
3. 敢下狠手。
4. 能利用规则坑反制敌人。
5. 能通过第二列字找到生路。

当前弱点：

1. 肉身一般。
2. 无正式身份。
3. 被官方注销。
4. 被黑市、裁钟、城防系统关注。
5. 污染未控。
6. 不能长时间正面硬刚武者或官方力量。
7. 黑票使用越多越危险。

短期成长方向：

1. 获得第一张真正可主动使用的罚单。
2. 掌握低阶规则反制手段。
3. 进入黑市资源链。
4. 得到临时隐藏身份或假权限。
5. 开始接触基础武道或高武资源。

# 6. 战力等级表

## 6.1 当前阶段定位

当前还不是传统高武打怪阶段。

第一卷重点是规则生存、身份危机、黑市入局、污染代价。

武道体系可以逐步露出，但不能过早变成纯战力碾压。

## 6.2 暂定武道等级

淬体：基础肉身强化，适合下城区训练营、低阶武者。

开脉：感知与爆发提升，可在规则副本中承受更多污染或冲击。

燃血：短时间爆发强，开始接触异常遗留物。

镇域：能在小范围内稳定自己的规则判断，不易被低级规则误导。

灾门：能短暂对抗高等级静默事件。

王座：后期等级，暂不展开。

## 6.3 规则权限等级

白票：最低层权限，能被动参与规则事件。

灰票：能读取或接触低层规单。

黑票：能看见部分第二列字，但代价很重。江彻当前为黑票状态。

血票：能主动生成一次规则惩罚或罚单。

金票：能改写低级局部规则。

王签：终局级权限，能制定或重写规则。

## 6.4 江彻当前战力状态

肉身：普通偏上，靠狠和反应弥补。

规则判断：强。

正面对抗低阶黑市打手：可赢，但不能轻松。

正面对抗训练有素官方人员：劣势。

正面对抗异常体：不能硬拼，只能利用规则。

面对顾承霄：不能硬刚。

面对白手套：暂时只能躲或借规则反制。

## 6.5 成长节奏

第一卷前20章：

规则判断为主，武道只露底。

20到40章：

正式接触低阶武道训练，获取低层身份或资格。

40到60章：

进入武考或官方筛选线，完成第一次阶层跃迁。

后续再逐步强化高武升级流。

# 10. 防止长篇崩盘的维护规则

每写3到5章更新一次：

1. 新增角色。
2. 新增规则。
3. 新增伏笔。
4. 已回收伏笔。
5. 主角获得与损失。
6. 反派推进。
7. 污染程度。

每写10章复盘一次：

1. 本阶段核心爽点。
2. 本阶段最大秘密。
3. 本阶段角色变化。
4. 本阶段读者最想追的问题。
5. 下一阶段必须兑现的内容。

绝对避免：

1. 连续刷副本但主线不推进。
2. 每章都靠第二列字硬解。
3. 江彻每次都轻松赢。
4. 配角只负责解释。
5. 反派只负责找死。
6. 世界观无限膨胀但不回收。
7. 规则越来越随意。

旧河公寓已经确认不是单纯等待正式开启的副本，而是可以通过门牌、住户回执、查房声、钟声、招租广告、水迹和住户关系提前影响现实。

新增概念：同住人。

404将齐子衡判为住户，并将江彻判为同住人。同住人需要履行住户关系义务，例如确认住户状态、回应敲门、接收回执。若同住人拒绝，住户可自行上门。

新增概念：临时签收点。

见雪诊所原本被旧河公寓判定为签收地址。江彻、苏见雪、唐九灯尝试将签收地点转移到南平旧体育馆。南平旧体育馆曾经是临时隔离点，也与旧河公寓试错队有关，因此可以短暂承载签收规则。

新增概念：临时筛查身份。

南平旧体育馆的基础淬体筛查系统可以在离线状态下为江彻建立特殊筛查档案。该档案让江彻的临时身份归属于旧体育馆，有效时间三小时，能短暂干扰旧河公寓对江彻的404同住人判定。

新增概念：隔离观察模式。

苏见雪利用江彻血样维持筛查系统体征刷新，使旧河公寓误以为江彻仍在旧体育馆。隔离观察模式可维持约48分钟，但并非绝对安全。

新增概念：地下钟室。

旧河公寓住户须知残页提到地下钟室。齐子衡称上一任江彻在钟下面。唐七灯遗留规单也提到地下钟室。

第二列字提示：

地下钟室不在地下。

它在所有敲错门的人下面。

新增机制：钟声与自动分房。

第16章末，旧河公寓钟声提前响起。第三声钟响后三分钟内，未登记者将自动分房。
