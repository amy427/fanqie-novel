新书处于立项孵化阶段。

旧书 `高武：违规者才配活着` 已停止常规推进，保留为归档和经验样本。

## 生产原则

1. 不自动发布。
2. 自动化只生成到发布前一步。
3. 每章必须给用户人工改稿清单。
4. 生成稿不能直接当最终稿。
5. 开篇优先追读、人物欲望、具体场景，不堆设定。
6. 所有“像AI水文”的段落必须标出并要求人工改写。

## 当前缺口

需要用户确定：

1. 题材方向。
2. 书名或暂定名。
3. 主角身份。
4. 核心卖点。
5. 前三章钩子。
6. 是否继续男频高武/规则恐怖，还是换赛道。
