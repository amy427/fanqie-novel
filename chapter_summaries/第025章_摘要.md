# 第025章_摘要

## 1. 章节标题

第025章 影子不算本人

## 2. 本章核心剧情

齐子衡影子里的404门牌实体化后，试图以“住户影子代返”的方式替齐子衡完成返房，并诱导江彻回应“同住人”确认关系。江彻没有直接答话，而是先确认齐子衡本人仍有疼痛、恐惧和自主回应，再用七灯铜扣和旧河公寓见证回执反向核验影子是否具备“自主认名反应”。影子能背出齐子衡姓名，却无法回答怕死、活人库状态和现实位置，只能反复以404住户身份作答。江彻借苏见雪在旧体育馆念出的体征刷新记录，以及顾承霄打开待同步档案留下的现场痕迹，把现实端活体记录与下层见证回执咬在一起，暂时压住回执签收和影子代返。代价是认线向锁骨和胸口分岔，钟背钥匙继续崩损，七灯铜扣上的“七灯”字迹变淡。章末，影子补登被卡住后，门缝灯光转为十三号线式冷白，一只白手套从404门内伸出，拿出见证回执副页，显示三年前见证人拒签后由白手套接管签收，要求江彻补齐签收。

## 3. 本章新增信息

1. 旧河公寓存在“住户影子可先行返房”的补登路径，但影子不得代住户认名。
2. 齐子衡影子能复制姓名和房号，却无法回答怕死、活人库状态、现实位置等自主认名核验项。
3. 见证回执可作为证据压住规则冲突，但不是罚单，使用会反噬江彻认线。
4. 苏见雪读出的旧体育馆体征刷新记录可与下层见证回执形成现实端核验冲突。
5. 三年前唐七灯拒签后，白手套接管了见证回执签收流程。

## 4. 本章推进的旧伏笔

1. 推进齐子衡被拖入404后会变成什么：本章确认404先转移到影子层，试图以影子代替本人返房。
2. 推进唐七灯拒签见证回执的旧案线：七灯铜扣和见证回执继续成为卡住错误签收的关键证据。
3. 推进白手套流程：白手套不只是列车检票者，还曾在三年前旧案中接管签收。
4. 推进顾承霄待同步档案线：顾承霄打开档案反被江彻借作现场核验痕迹。
5. 推进认线代价：江彻不用第二列字也会因强行压规则证据而加深污染。

## 5. 本章新增伏笔

1. 白手套持有的见证回执副页是否能与江彻手里的正页拼合。
2. 三年前“签收接管：白手套”具体意味着谁替谁收走了旧案证据。
3. 七灯铜扣继续使用后，唐七灯剩余认知会被消耗到什么程度。
4. 齐子衡影子被卡住后，是否会变成独立的404门内执行体。
5. 顾承霄现场听见旧体育馆记录后，是否会选择封存、销毁或利用该记录。

## 6. 本章角色状态变化

1. 江彻：保住见证回执未被签回，暂时卡住影子404代返，但认线向锁骨和胸口分岔，钟背钥匙继续崩损。
2. 齐子衡：本人仍未彻底写死为404，凭恐惧、疼痛和自主回答维持“未完成登记”，但影子已高度门牌化。
3. 苏见雪：在现实端配合江彻念出体征刷新记录，主动把旧体育馆终端记录卷入下层核验。
4. 顾承霄：打开待同步档案的行为被江彻反向利用，成为现场核验链的一部分。
5. 唐七灯：七灯铜扣被进一步消耗，残存认知风险加重。

## 7. 本章章末钩子

影子404被核验卡住后，404门内伸出一只白手套，拿出旧河公寓见证回执副页。副页显示三年前见证人拒签后，签收由白手套接管。白手套要求江彻补齐签收。

## 8. 下一章必须承接点

下一章必须直接承接白手套拿出见证回执副页，处理江彻如何在不补齐签收、不交出正页的情况下，弄清白手套当年接管签收的流程，并继续维持齐子衡未完成登记状态。
