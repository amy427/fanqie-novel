# 第029章_摘要

## 1. 章节标题

第029章 影子补票

## 2. 本章核心剧情

列车员在404门缝后的冷白车厢前递出现场复核补票单，要求齐子衡影子作为第八次样本陪同载体补票上车。江彻没有直接抢影子，而是逼列车员先说明乘车规则，抓住“同住人是关系不是票种”“登记不等于运送”“候车需持票即等于上车”等流程漏洞，将补票流程拖入复核问答。齐子衡本人被迫回答活人状态、恐惧反应和拒绝授权，苏见雪补充旧体育馆封存箱记录，证明影子无独立体征、血样、疼痛反馈和自主签名。江彻再用七灯铜扣和见证回执正页压住补票单，迫使列车员承认齐子衡影子只能待复核，不能先行上车，也不能代替本人返房。但第八次回收车厢仍保留陪同载体需求，问答单背面出现新乘客栏，显示上一任J-317已在第七节车厢候车。

## 3. 本章新增信息

1. 十三号线现场复核补票单可被证据链逼成复核问答单。
2. 齐子衡影子当前被判定为待复核对象，不得先行上车，不得代替本人返房。
3. 第八次回收车厢需要陪同载体，但需求来源和签发人仍未出示。
4. 上一任J-317已在第七节车厢候车。

## 4. 本章推进的旧伏笔

1. 推进“上一任J-317是谁”线，上一任J-317从线索变成现场候车对象。
2. 推进第八次回收流程，确认它不是单纯样本采集，还涉及陪同载体。
3. 推进404同住关系，确认同住关系会被十三号线尝试转换为陪同责任。
4. 推进齐子衡影子线，影子不能完全代表齐子衡本人。

## 5. 本章新增伏笔

1. 第八次回收车厢为什么不能让江彻一个人上车。
2. 上一任J-317在第七节车厢里的状态是真人、残影、记录，还是旧流程保留物。
3. 列车员在现场复核中为什么能压住白手套。
4. 齐子衡影子待复核后会被404、公寓或十三号线如何处置。

## 6. 本章角色状态变化

1. 江彻继续失血，认线压向心口，但成功阻止齐子衡影子被先行补票上车。
2. 齐子衡本人明确表达拒绝授权，并意识到自己可能被拆成可代签部件。
3. 顾承霄继续以只读字段协助江彻，但被城防终端追责。
4. 苏见雪通过旧体育馆封存箱补充证据，继续维持现实端锚点。

## 7. 本章章末钩子

上一任J-317出现在第八次样本陪同席的新乘客栏中，座位号为第七节车厢，并用江彻自己的声音提醒：“别让他一个人上车。”

## 8. 下一章必须承接点

第030章必须直接承接第七节车厢和上一任J-317候车状态，解释“别让他一个人上车”指向江彻、上一任J-317，还是第八次回收流程中的另一个对象。
