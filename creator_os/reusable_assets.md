# Reusable Assets

## 标题模板

1. 物件 + 动作：影子补票、黑票验人、门牌认主。
2. 规则反常：活人名单上没有你、医生说你快不像人了。
3. 现场命令：别让他一个人上车。
4. 身份翻牌：上一任J-317。

## 爽点模板

1. 把对方的流程名词拆成法律/规则意义上的漏洞。
2. 把“默认同意”反打成“授权来源空白”。
3. 把“候车”反打成“事实进入车厢”。
4. 把“影子意愿”反打成“无活体反应，无自主签名”。

## 短视频/简介句

1. 十三号线想让他的同住人影子补票上车，他先问影子算不算活人。
2. 别人怕列车员催票，江彻逼列车员把补票单改成问答单。
3. 第七节车厢里坐着上一任自己，还用他的声音提醒：别让他一个人上车。
