# Weekly Review Spec

Current version date: 2026-05-06

## Purpose

Run once per week to check whether the serial is advancing correctly.

## Review Inputs

1. `chapters/`
2. `chapter_summaries/`
3. `daily_output/`
4. `continuity/`
5. `feedback/`
6. `decision_log/`
7. `automation/run_index.md`
8. `creator_os/`

## Required Output

Create:

```text
weekly_review/YYYY-MM-DD_周复盘.md
```

## Review Sections

1. 本周新增章节
2. 本周归档状态
3. 伏笔推进情况
4. 拖延风险
5. 读者反馈/数据变化
6. 标题表现
7. 下周最重要的三个目标
8. 是否需要更新 NOVEL_OS 或 CREATOR_OS

## Priority Rule

Weekly review should improve the next week of writing. It should not become bureaucracy.
