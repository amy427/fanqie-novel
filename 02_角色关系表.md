# 02_角色关系表

## 1. 江彻

主角。

身份：下城区七号线夜班临时巡检，现已被官方活人库注销。

当前目标：

1. 活下去。
2. 查清三年前J-317注销案。
3. 找到上一任自己。
4. 弄清十三号线与旧河公寓的关联。
5. 查白手套、裁钟、静默处置科。
6. 反向夺取规则权限。
7. 解决404同住人关系。
8. 控制认线污染。
9. 利用残缺罚单或地下钟室下层线索推进旧河公寓事件。

角色写法：

冷静，但不是无情。

狠，但不是无脑。

会算账，会记仇，会利用敌人的规则坑敌人。

## 2. 唐九灯

身份：九灯钟表铺管事，黑市情报贩子，规则交易中介。

标签：嘴上松散，算账极清，懂规单，熟下城区黑市。

与江彻关系：交易型盟友，目前因唐七灯线与江彻绑定更深。

当前立场：

1. 帮江彻，因为江彻活着对他有价值。
2. 想借江彻咬出十三号线、旧河公寓、裁钟背后的线。
3. 不是纯好兄弟，不能写成无条件帮忙。
4. 与旧河公寓存在私人旧账。
5. 师兄唐七灯被困于地下钟室相关流程。

当前关键物：

1. 唐七灯页副本。
2. 唐七灯怀表。
3. 烧焦红绳。
4. 半张旧河公寓住户须知。

## 3. 苏见雪

身份：医生，静默旧案追查者，疑似与三年前J-317案有关。

标签：冷静、专业、直接，不安慰人，只判断还能不能活。

与江彻关系：专业合作，临时共谋。

1. 追查三年前那批被定义成替身的人，是否还有原人。
2. 不站裁钟，也不站静默处置科。
3. 愿意帮江彻压制认线，但不是无条件保护。
4. 留守南平旧体育馆，维持临时签收点、封存箱、定位钉和隔离观察模式。

当前风险：

1. 旧体育馆隔离观察模式可能因临时筛查卡被剪受影响。
2. 旧河公寓正式开启后，苏见雪可能承受封存箱或404门牌反扑。
3. 城防或顾承霄可能接近旧体育馆。

注意：不要过早写成感情线。
