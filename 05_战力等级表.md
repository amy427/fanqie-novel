# 05_战力等级表

## 1. 当前阶段定位

当前还不是传统高武打怪阶段。

第一卷重点是规则生存、身份危机、黑市入局、污染代价。

武道体系可以逐步露出，但不能过早变成纯战力碾压。

第12到20章已正式引出：

1. 基础淬体筛查。
2. 气血值。
3. 特殊筛查档案。
4. 普通人体质难以承受认线。
5. 武道不是为了装强，而是为了让身体多扛几层污染和错误。

---

## 2. 暂定武道等级

### 淬体

基础肉身强化。

适合下城区训练营、低阶武者。

功能：

1. 提升皮肉、神经、反应、气血稳定度。
2. 在规则副本中多争取一点纠错时间。
3. 增强污染承载能力。

### 开脉

感知与爆发提升。

可在规则副本中承受更多污染或冲击。

### 燃血

短时间爆发强，开始接触异常遗留物。

### 镇域

能在小范围内稳定自己的规则判断，不易被低级规则误导。

### 灾门

能短暂对抗高等级静默事件。

### 王座

后期等级，暂不展开。

---

## 3. 规则权限等级

白票：最低层权限，能被动参与规则事件。

灰票：能读取或接触低层规单。

黑票：能看见部分第二列字，但代价很重。江彻当前原始状态为黑票相关。

血票：能主动生成一次规则惩罚或罚单。江彻当前尚未确认拥有血票，但已获得残缺罚单和一次低阶违规判定，可能是血票雏形。【待确认】

金票：能改写低级局部规则。

王签：终局级权限，能制定或重写规则。

## 4. 江彻当前战力状态

肉身：普通偏上，靠狠和反应弥补。

当前气血值：0.74。

下城区普通成年男性基准：0.68。

低阶淬体入门参考：1.20。

基础评价：普通，但比普通基准略高。

规则判断：强。

正面对抗低阶黑市打手：可赢，但不能轻松。

正面对抗训练有素官方人员：劣势。

正面对抗异常体：不能硬拼，只能利用规则。

面对顾承霄：不能硬刚。

面对白手套或列车员：暂时只能躲、借规则反制或制造流程冲突。

### 5.2 残缺罚单

第20章状态：

不是普通战力，不是直接攻击技能。

2. 干预登记簿。
3. 作为旧河公寓下层流程权限之一。

限制：【待确认】。

功能：

## 6. 江彻当前污染状态

认线已明显恶化。

已知表现：

1. 手背虎口黑线。
2. 黑线过腕。
3. 进入钟心后黑线越过小臂三寸。
4. 使用罚单、撕编号、干预登记簿都会加重认线。

1. 身体承受能力不足。
2. 临时身份受损。
3. 旧河公寓、十三号线、裁钟同时追缴。
4. 若无法进入高武体系，后续可能承受不了更深规则权限。

已达成：

### 第21到40章

建议方向：

1. 正式接触低阶武道训练。
2. 获取低层身份或资格。
3. 利用旧河公寓收益换取黑市资源。
4. 处理认线恶化。
5. 与顾承霄达成观察或临时交易。
6. 争夺下城区武考资格或进入基础淬体训练。

### 第41到60章

建议方向：

1. 进入更高层筛选系统。
2. 完成第一次阶层跃迁。
3. 让规则能力和武道肉身开始组合。
4. 面对正式裁钟成员或顾承霄同级人物。

## 8. 当前不允许写崩的地方

1. 江彻不能突然变成高武强者。
2. 残缺罚单不能当万能攻击技能。
3. 第二列字不能无成本解决全部问题。
4. 认线不能只写设定，要持续带来身体和行动代价。
5. 高武训练必须从承受污染、活下去的需求自然推进。
6. 顾承霄、裁钟、列车员不能无脑降智。
