# Monthly Interview

每月让 agent 采访你一次。建议语音回答，转成文字后放进 `inbox/`。

## 本月复盘

1. 这个月最重要的三件事是什么？
2. 哪件事改变了你对自己的理解？
3. 哪件事改变了你对某个人的理解？
4. 哪个项目有实质进展？哪个项目卡住了？
5. 这个月你反复提到、反复思考的主题是什么？

## 自我画像

1. 这个月你更确定自己是一个什么样的人？
2. 你发现自己有什么以前没意识到的偏好？
3. 你最近做决定时，最常用的判断标准是什么？
4. 你最近逃避了什么？
5. 你最近真正想要的是什么？

## 关系记录

1. 这个月和父母、伴侣、朋友发生了哪些值得记住的互动？
2. 有没有一句话、一个场景，以后你可能会想重新回看？
3. 有没有没说出口的感谢、歉意、担心或承诺？

## 身后安排与长期遗憾

1. 如果这个月是你人生最后一个月，你会觉得哪些事还没交代清楚？
2. 哪些文件、账号、资产、作品需要被整理？
3. 你希望重要的人如何理解你这个月的选择？
4. 你不希望别人误解你的哪件事？

## 下月行动

1. 下个月最值得推进的一件事是什么？
2. 哪些事可以停止做？
3. 哪些信息需要继续收集？
